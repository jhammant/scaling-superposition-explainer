# Superposition Yields Robust Neural Scaling — Interactive Explainer

Static GitHub Pages explainer for the paper [Superposition Yields Robust Neural Scaling](https://arxiv.org/abs/2505.10465) by Yizhou Liu, Ziming Liu, and Jeff Gore.

## What it covers

- The representation bottleneck behind LLM scaling
- Weak vs strong superposition
- A small interactive toy simulator for overlap-driven error
- The paper's LLM evidence and implications

## Run locally

```bash
python3 -m http.server 8766
```

Then open <http://127.0.0.1:8766/>.

## Deploy

This repo is intended to deploy directly from `main` using GitHub Pages.
